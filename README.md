# ❌⭕ Tic Tac Toe Game

A simple and interactive Tic Tac Toe game built using HTML, CSS, and JavaScript.

## Features

* 🎮 Two-player gameplay (X and O)
* 🏆 Winner detection
* 🤝 Draw detection
* 🔄 New Game button
* ♻️ Reset Game button
* 📱 Responsive user interface

## Technologies Used

* HTML
* CSS
* JavaScript

## How to Play

1. Player O starts the game.
2. Players take turns selecting empty boxes.
3. The first player to get three symbols in a row, column, or diagonal wins.
4. If all boxes are filled and no winner is found, the match ends in a draw.
